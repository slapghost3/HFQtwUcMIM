# 图书馆管理系统简介 python245

## 项目概述

图书馆管理系统是基于Python语言，采用Flask框架和MySQL数据库构建的一套软件。系统旨在实现图书馆的数字化管理，提高图书管理的效率和便捷性。

## 技术栈

- **后端语言与框架**：Python 3.x，Flask
- **数据库系统**：MySQL
- **开发工具**：PyCharm
- **其他工具**：Navicat（数据库连接工具）

## 功能模块

- **用户端**：
 - 用户登录与注册
 - 图书信息查询
 - 图书借阅申请

- **管理端**：
 - 图书信息增加、修改、删除
 - 用户管理
 - 图书采购申请管理
 - 数据可视化

## 系统角色

- **用户**：可登录、查询图书、申请借阅、注册账户
- **管理员**：管理图书信息、查看用户信息、处理图书采购申请

## 运行环境

- **操作系统**：兼容 Windows、Linux
- **数据库系统**：MySQL

## 部署步骤

1. 使用PyCharm导入项目。
2. 安装项目所需的依赖包。
3. 配置MySQL数据库。
4. 运行项目。
